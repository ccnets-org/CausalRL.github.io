Causal Reinforcement Learning

A powerful and robust RL algorithm that allows for network customization and the ability to learn causal relationships.

Introduction

Causal RL is an innovative Reinforcement Learning framework that utilizes three networks: Actor, Critic, and Reverse Environment, to learn the causal relationships between states, actions, and values while maximizing cumulative rewards. This introduction describes key features for understanding Causal RL.

Basic Usage

# main.ipynb
import torch
from utils.setting.env_settings import analyze_env
from causal_rl import CausalRL

device = torch.device("cuda:0" if (torch.cuda.is_available() and ngpu > 0) else "cpu")

env_config, rl_params = analyze_env(env_name = "Humanoid-v4")

with CausalRL(env_config, rl_params, device, use_print = False, use_wandb = True) as causal_rl:
    causal_rl.train(resume_training = False, use_eval = False, use_graphics = False)
    causal_rl.test(max_episode = 100, use_graphics = False)
